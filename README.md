# MCQ Quiz Application

A minimal and responsive Multiple-Choice Quiz Application built using HTML, CSS, and JavaScript.  
The app generates 15 random questions from a larger question bank and provides an interactive quiz experience with a clean, simple UI.

---

## Features

- 15 randomized questions per attempt  
- Start screen with instructions  
- Clean and distraction-free quiz interface  
- Clickable option-based answers  
- Automatic scoring  
- Detailed result summary page  
- Fully client-side (no backend needed)

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
